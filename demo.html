Minimal HTML markup for musical chord notation with colored visual coding (riso-colors). Click any chord to see guitar fingering diagram.

Features

  • Color-coded chords: Each root note gets unique color (C=red, D=yellow, E=green, etc.)
  • Minor chord distinction: Subtle red tint background + thicker color band
  • Interactive diagrams: Click chords to see guitar fingering positions
  • Alternative chords: Use alt attribute for dashed border
  • Half-width chords: Use half attribute for passing chords that share one slot
  • Technique icons: Use oct="1|2" and shell="1|2" to show 24x24 icons (shell major/minor inferred from chord name)
  • Legacy indicators: Use up/down attributes to show octave direction arrows (â†‘/â†“)
  • Inline chord notes: Add a child tag like <em> for small subtext inside the box
  • Split bass box: Use bass="..." to show a stacked lower chord area
  • Separators: Use <sep> for visual breaks between chord groups
  • Compact printing: Optimized layout for cheat sheets

Basic Usage

HTML Structure

<song>
    <h2>Song Title</h2>
    <song-meta>
        <span>Toneart: Am</span>
        <span>Tempo: 120 BPM</span>
    </song-meta>
    
    <h3>Section Name</h3>
    <p>Optional notes</p>
    <chord>Am</chord>
    <chord>F</chord>
    <chord>C</chord>
    <chord>G</chord>
</song>

Special Attributes

<!-- Alternative chord (dashed border) -->
<chord alt>Em</chord>

<!-- Half-width passing chord -->
<chord half>C</chord>

<!-- Technique icons (24x24) -->
<chord oct="1">Am</chord>
<chord oct="2">Am</chord>
<chord shell="1">Am</chord>  <!-- minor icon inferred -->
<chord shell="2">E</chord>   <!-- major icon inferred -->

<!-- Legacy: octave direction indicators -->
<chord up>Am</chord>
<chord down>Am</chord>

<!-- Inline note shown on small second line -->
<chord>C <em>Play scale up</em></chord>

<!-- Split chord with dedicated bass box -->
<chord bass="A">C</chord>

<!-- Half-width split passing chord -->
<chord half bass="A">C</chord>

<!-- Separator slash -->
<sep></sep>

<!-- Specific voicing/fret (for metadata) -->
<chord data-voicing="barre" data-fret="5">Dm</chord>
